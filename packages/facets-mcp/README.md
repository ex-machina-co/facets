# @ex-machina/facets-mcp

[![npm](https://img.shields.io/npm/v/@ex-machina/facets-mcp)](https://www.npmjs.com/package/@ex-machina/facets-mcp)

An MCP server that exposes facet operations to AI coding assistants. Built on `@ex-machina/facets` and the [Model Context Protocol SDK](https://github.com/modelcontextprotocol/typescript-sdk).

> **Status**: Early development (v0.1.0). APIs may change.

## Usage

The server runs over stdio:

```sh
bunx @ex-machina/facets-mcp
```
