# Facets

A package manager and toolkit for facets — modular skills, agents, commands, and tools that extend AI coding assistants.

## Packages

| Package                                     | NPM                      | Description                              |
|---------------------------------------------|--------------------------|------------------------------------------|
| [Facets CLI](packages/facets/README.md)     | `@ex-machina/facets`     | Core library and CLI                     |
| [MCP Server](packages/facets-mcp/README.md) | `@ex-machina/facets-mcp` | MCP server that exposes facet operations |

## Development

### Prerequisites

- [Bun](https://bun.sh) (v1.3.10+)

### Setup

```sh
# Install dependencies
bun install

# Run lint, typecheck, build, and tests
bun check
```
